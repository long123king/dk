## [1.0.0] - TBD

### Added
- Initial release of dk WinDbg extension
- Enhanced refactored version of tokenext
- Debugger Data Model support
- Time Travel Debugging support
- SVG visualization generation
- Comprehensive command set for debugging Windows internals
- Commands for analyzing:
  - Memory pages
  - Heap allocations
  - Processes and threads
  - Tokens and security
  - Handles and objects
  - Pool allocations
  - And more...
